Sobre mí ▼

Soy futura Técnica Universitaria en Desarrollo de Aplicaciones Informáticas, a punto de finalizar mi formación (solo me queda un examen final para obtener el título).

Desde agosto de 2024, trabajo en la UNICEN, en el área de informática, donde participo activamente en el desarrollo y mantenimiento de sitios web institucionales, buscando siempre optimizar la experiencia del usuario y aportar soluciones eficientes.

Me considero una persona apasionada por la tecnología y el aprendizaje continuo, con ganas de enfrentar nuevos desafíos y crecer profesionalmente.

Disfruto trabajar en equipo, colaborando y comunicándome de manera efectiva para lograr objetivos comunes y entregar resultados de calidad.

Además, me interesa mantenerme actualizada en las últimas tendencias y herramientas de desarrollo, combinando creatividad y lógica para crear soluciones que marcan la diferencia.

Proyectos destacados

Proyecto de Historia Clínica

Desarrollo de un sistema de historia clínica utilizando PHP y MySQL. El proyecto sigue el patrón de diseño MVC (Modelo-Vista-Controlador) y utiliza templates `.tpl` para la representación de la interfaz de usuario. Permite gestionar de manera eficiente las historias clínicas de los pacientes, con funcionalidades para agregar, editar y consultar registros médicos.

Proyecto vinoteca en MVC y API REST

Trabajo colaborativo en grupo realizado para la materia Web 2 de TUDAI. Proyecto de vinotecas realizado en dos partes: un sistema MVC y una API REST, ambos diseñados para mostrar métricas de negocio en tiempo real. El dashboard es responsive y construido con React y Chart.js para una visualización dinámica y clara de los datos.

Desarrollo de Páginas Personalizadas en WordPress

He trabajado en la personalización de sitios web utilizando WordPress, creando temas personalizados para satisfacer las necesidades de cada cliente. Esto incluyó la integración de funcionalidades específicas mediante la creación de Custom Post Types (CPT) y la modificación de plugins existentes para lograr un rendimiento óptimo. Me aseguré de que todos los sitios fueran completamente responsivos, brindando una excelente experiencia de usuario en dispositivos móviles.

Creación y Personalización de Custom Post Types (CPT)

He diseñado y desarrollado Custom Post Types (CPT) para permitir a los clientes gestionar diferentes tipos de contenido en WordPress de manera eficiente. Esto incluyó la creación de taxonomías personalizadas, la integración de campos personalizados con Advanced Custom Fields (ACF) y la creación de plantillas personalizadas para cada tipo de contenido, mejorando la administración y la visualización de la información en el sitio web.

Desarrollo de Plugins Personalizados en WordPress

A lo largo de mi experiencia en WordPress, he desarrollado plugins personalizados para resolver necesidades específicas de los proyectos. Desde la creación de funcionalidades para la gestión de contenido, hasta la integración de sistemas externos, estos plugins fueron diseñados para ser altamente optimizados, seguros y fáciles de mantener, contribuyendo a la mejora de la eficiencia y la experiencia de usuario.

Análisis de Datos con Apache Superset

He trabajado con Apache Superset para el análisis de datos en proyectos complejos. Utilizando esta herramienta de visualización y exploración de datos, he sido capaz de crear dashboards interactivos y visualizaciones dinámicas que han permitido a los clientes tomar decisiones informadas basadas en datos. Este análisis incluyó la creación de gráficos personalizados, filtros y la integración de diversas fuentes de datos.
